Battle Ship

Battle Ship Instructions

1. The first input would set each target within the game grid

2. The next inputs should be similar to that of a chess board
Ex. C3.

3. Once all targets have been placed, another player can attack
until the value -1 has been input.

4. The game board will thne be displayed with each target hit is
displayed as an X, and each target miss as an O, and everything
else as an underscore _.

5. Then it will print whether the attacking player wins Attacking
Player Wins, or the defending player wins Defending Player Wins.
Printing the number of hits, misses, and remaining targets.
